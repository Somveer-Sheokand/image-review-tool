# Image Review Tool

A simple **CSV-based image review web tool** for quickly reviewing and classifying images as **Good** or **Bad**.

Built using **HTML, CSS, and JavaScript** and runs entirely in the browser.

---

## Features

* Upload CSV file with image URLs
* Review images one by one
* Mark images as **Good** or **Bad**
* Keyboard shortcuts
* Progress bar
* Review statistics
* Export results as CSV

---

## CSV Format

Your CSV file must contain **two columns**:

```
id,url
DL30SR6596,https://example.com/image1.png
AP95AM2799,https://example.com/image2.png
AS97ZE5377,https://example.com/image3.png
```

| Column | Description      |
| ------ | ---------------- |
| id     | Image identifier |
| url    | Direct image URL |
